# user-service

A Spring Boot microservice showcasing core backend engineering skills:

- **REST API design** — `@RestController`, layered architecture (Controller → Service → Repository)
- **Spring Data JPA** — ORM with PostgreSQL, repository pattern, `@Transactional` operations
- **Event-driven messaging** — Kafka producer integration for async user-deleted events
- **Exception handling** — Custom exceptions with `@ControllerAdvice` global handler
- **Clean code** — Lombok, Builder pattern, DTO separation, structured logging with SLF4J
- **Java 21 + Spring Boot 4.x** — Modern stack with Maven build
